# SotR-ICUA-2026
This repository contains the code used to conduct all data processing steps before manual labelling in the paper: 'Sounding out the river: and end-to-end framework for monitoring bioacoustic events and sediment movement in freshwater soundscapes' submitted to the ICUA 2026. Pre-print available at: https://doi.org/10.32942/X2QT0D
